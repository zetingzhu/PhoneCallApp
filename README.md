## Phone Call App

This is a demo shows how to make your app as the default phone call app and manage phone calls(Android 6.0+) and also listen phone call states.

这是一个示例项目演示如何将应用设置为默认电话应用并对电话接打等进行管理，同时也演示了如何监听来电状态并进行一些操作。
